# 衛星画像を用いた軽石（Pumice）自動検知訓練パイプライン

## プロジェクト概要
本プロジェクトは、小笠原諸島の海底火山噴火等に伴う軽石流出活動を衛星画像（GCOM-C）から自動検知するためのモデル訓練環境です。
正常な海面と軽石を精密に識別し、沿岸部への影響評価や船舶の航行安全に寄与することを目的としています。

## 技術的ポイント
- シングルクラス最適化: 「軽石」のみを対象とすることで、海面ノイズや雲との誤検知を抑制。
- マルチスペクトル解析: 単なるRGB画像ではなく、軽石特有の分光反射特性を前処理で活用。
- 高解像度訓練: 衛星画像内で数ピクセル単位で現れる軽石を捉えるため、入力解像度を1280pxに最適化。

## 成果
- 突発的な大規模軽石漂流における早期発見および、経時的な流動解析パイプラインの構築に貢献。
